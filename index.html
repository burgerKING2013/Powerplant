📘 POWERPLANTΦ FULL TUTORIAL

Created by: Atila

PowerPlantΦ is a custom programming language featuring a Virtual Machine execution model, a built-in Physics Engine, and complex variable handling, all running inside C++.

1. Variables & Assignment

Store numbers or strings into memory using the assignment arrow.

α ← 5
β ← 10
variable ← expression

2. Print System (π)

Loads values onto the stack, executes math if needed, and prints the result.

π 5
π α + β
π "hello"

3. Math & Logic

  • + Addition | - Subtraction
  • * Multiplication | / Division
  • √ Square Root (Syntax: π √ 25)

4. Control Flow

If Statement (ε)

Runs a block if the condition is non-zero.

ε α
    π "ok"
/ε

While Loop (ω)

Repeats execution while the condition is true.

ω α
    π α
    α ← α - 1
/ω

5. Functions (θ / ξ)

Define reusable code with θ and call it with ξ.

θ f
    π 999
end
ξ f

6. Physics Engine

Force: F = m × a | π F 10 2

Energy: E = 0.5 × m × v² | π E 5 3

Velocity: v = u + a × t | π v 10 2 3

7. System Architecture

SOURCE CODE → TOKENIZER → COMPILER → BYTECODE → VM → OUTPUT

The VM utilizes a Stack System for operations and a Globals Map for memory management.
